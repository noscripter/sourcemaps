### Installing Compass

If you would like to install Compass via bundler just run the command:

```bash
$ bundle install
```

**Without Bundler**

If you would like to install Compass w/out bundler run the command:

```bash
$ gem install compass
```

### Install NPM Modules

```bash
$ npm install
```

### Running Project

```bash
$ grunt
```
